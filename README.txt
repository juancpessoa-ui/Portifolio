
# Portfólio Profissional Web – Desenvolvedor Júnior

# Objetivo 
Desenvolver, no prazo de até 2 semanas, uma página de portfólio profissional responsiva utilizando HTML e/ou Markdown, com deploy no GitHub Pages, contendo informações pessoais, habilidades, projetos e contato, com o objetivo de apresentar o perfil profissional para recrutadores.

* Específico: Criar portfólio com seções obrigatórias
* Mensurável: Página publicada e acessível online
* Atingível: Uso de tecnologias básicas (HTML/CSS)
* Relevante: Auxiliar na entrada no mercado de trabalho
* Temporal: Entrega em até 10 dias

# Patrocinador
Projeto acadêmico (Senai)

#  Escopo
### Inclui:
* Desenvolvimento de página HTML ou Markdown
* Estrutura com header, seções e footer
* Publicação via GitHub Pages
### Não inclui:
* Backend
* Banco de dados
* Sistemas complexos

### Stakeholders:
* Desenvolvedor Juan
* Professor/avaliador
* Recrutadores

# Riscos
* Falta de conhecimento técnico
* Problemas no deploy
* Layout não responsivo

# Entregáveis
* Repositório GitHub
* Página publicada (GitHub Pages)
* Código HTML/Markdown
